# Medical Form Builder

Welcome to the **Medical Form Builder**! This project helps users create customizable medical forms with ease and efficiency using cutting-edge technologies.

## Features
- **Built with React**: A powerful JavaScript library for building user interfaces.
- **TypeScript**: Enhances JavaScript by adding static types, making your code more robust and maintainable.
- **Vite**: A modern build tool that offers a fast and lean development experience.
- **Drag-and-Drop Functionality**: Create forms intuitively by dragging and dropping elements.
- **Tailwind CSS**: A utility-first CSS framework for designing responsive and modern user interfaces.
- **Zustand**: A small, fast, and scalable state management solution.

## Getting Started
### Prerequisites
Make sure you have the following installed:
- Node.js (version 14 or higher)

### Installation
1. Clone this repository:
   ```bash
   git clone https://github.com/AshGov07/Medical_Form_Builder.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Medical_Form_Builder
   ```
3. Install the dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm run dev
   ```
